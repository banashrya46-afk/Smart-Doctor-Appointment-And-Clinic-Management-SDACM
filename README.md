## Project Overview

The Smart Doctor Appointment & Clinic Management System is a healthcare application that allows patients to book doctor appointments based on their location, clinic schedule, and doctor availability.
This project focuses on performing manual functional testing of the SDACM system to ensure that the core booking workflow works correctly and provides a smooth user experience.
Testing activities in this project were documented using Microsoft Excel, including test cases, defect reports, and test execution results.

## Module Tested

The following modules were considered during manual testing:

1. User Registration & Login & Logout
2. Forgot Password
3. Find Doctor Page & Show Result & Provider ProfileDetials Page
4. Doctor Availability & Clinic Schedule
5. Online Medicine Order
6. Online Lab Test Booking
7. Surgeries Booking
8. Consult with a Doctor
9. Payment Page & Profile Management
